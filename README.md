## Getting Started

1. Go to [Remix](https://remix.ethereum.org/)
2. Paste the code from `SimpleStorage.sol` into a new file in Remix
3. Hit `Compile` 
4. Hit `Deploy` 


For a more in depth blog, [read here](https://docs.chain.link/docs/deploy-your-first-contract/)
